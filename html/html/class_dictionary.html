Dictionary::Dictionary ( )

This is the constructor of the Dictionary It reads all the words from the "words_alpha.txt" file and adds them to the corresponding set depending on their lengths. the Dictionary class can be seen as a partition of a normal Dictionary where each word belongs to one set and the specific set to which it belongs depends on the length of the word
